## 产品方向

BookVoice Studio 的长期目标不是堆格式转换能力，而是让用户稳定地把电子书生产成可听、可管理、可收藏的有声书资产。

当前最重要的体验闭环是：

```text
导入电子书 -> 识别书名/章节 -> 调整朗读参数 -> 转换 -> 输出到书名目录 -> 可复听/可排错/可重试
```

## 核心原则

- 先保证 EPUB 质量，再扩展 TXT/DOCX/MOBI/AZW3/PDF 等格式。
- 先保证章节顺序和输出结构，再追求更多高级参数。
- 普通用户默认一键可用，资深用户可以展开高级设置。
- 界面展示摘要，日志保留细节。
- 所有失败都要有可理解的原因和下一步动作。

## 近期建议

下一阶段建议优先实施：

1. 自动按书名创建输出文件夹。
2. EPUB 元数据和章节解析优化。
3. 语速、音量、音调设置。
4. 进度显示、打开输出目录、失败重试。
